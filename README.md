# Personalized Webpage

A simple interactive webpage built with HTML, Tailwind CSS and JavaScript.

---

## What it does

The user enters their name and age, and the webpage:

- Displays a personalized greeting
- Shows their age in months
- Checks if they are old enough for adult content
- Displays a motivational quote five times
- Saves their information using localStorage so it persists after a page refresh

---

## Built With

- **HTML** — structure of the webpage
- **Tailwind CSS** — styling and responsive design
- **JavaScript**

---

## Features

- Form to collect user name and age
- localStorage to save and retrieve user data
- if/else statement to check user age
- Template literals for personalized greeting
- Function to calculate age in months
- Loop to display a motivational quote five times
